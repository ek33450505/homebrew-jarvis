# homebrew-jarvis

> **ARCHIVED** — JARVIS has been retired. Personal assistant agents now run via Anthropic RemoteTrigger + cron. This Homebrew tap is no longer maintained.

Homebrew tap for [JARVIS Personal Assistant](https://github.com/ek33450505/jarvis).

## Install

```bash
brew tap ek33450505/jarvis
brew install jarvis
bash $(brew --prefix jarvis)/install.sh
```

## What it installs

8 pa-* Claude Code agents with launchd scheduling for macOS — morning briefings, email triage, Jira standups, EOD summaries, weekly reviews, and more.

See the [jarvis repo](https://github.com/ek33450505/jarvis) for full documentation.
